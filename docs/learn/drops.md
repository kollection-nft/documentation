# Drops

Collection drops are an exciting feature on Kollection. Participating collections will do their minting live on Kollection!

A slider on our [Homepage](https://kollection.app) and the [Drops overview page](https://kollection.app/drops) will inform you about upcoming collections!

Get all the necessary information to learn everything that you need to know about the collection. The number of items, the price, the minting date example graphics and a lot more!
