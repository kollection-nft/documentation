<h1 className={"openapi__heading"}>Kollection Public API</h1>



This is the Kollection public API. In the interest of helping developers build open, transparent, and connected Web3 experiences on Koinos, we are providing some of our marketplace data free of charge. <br />If you use this data, we do require you to provide attribution to Kollection in your app or on your site. When using this API data, you should link to the Kollection website where it makes sense to do so. Our [brand guidelines](https://github.com/kollection-nft/branding) repository provides images that can be used.<br /><br />API results are cached at the CDN level - results could be up to 60 seconds old at any point in time.<br /><br />Presently there is only one API method available but more could be added in the future.
